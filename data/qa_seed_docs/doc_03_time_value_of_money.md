# Time Value of Money and Compounding

The core principle: **a dollar today is worth more than a dollar in the future** because money can earn returns.

## Key formulas

- **Future value**: FV = PV * (1 + r)^n
- **Present value**: PV = FV / (1 + r)^n

Where:
- PV = present value
- FV = future value
- r = rate per period
- n = number of periods

## Practical meaning

- Compounding rewards long horizons.
- Discounting translates future cash into today's dollars.
- Small differences in rate assumptions can materially change valuation estimates.

## Rule of 72

Approximate doubling time in years is `72 / annual return (%)`.

Examples:
- 6% -> about 12 years
- 9% -> about 8 years

## Common Q&A snippets

- **Q: Why does starting early matter so much?**  
  A: Early years compound for longer, creating non-linear growth.

- **Q: What is a discount rate in plain language?**  
  A: The return required to make future cash flows attractive relative to risk.
